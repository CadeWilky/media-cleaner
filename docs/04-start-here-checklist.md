# Start Here Checklist

1. Create the Expo project.
2. Copy `CLAUDE.md`, `.claude/agents/`, and `docs/` into the project root.
3. Open the project in Claude Code.
4. Run `/agents` so Claude loads the project subagents.
5. Run the interview prompt from `docs/01-spec-interview-prompt.md`.
6. Review the generated refined spec.
7. Run the planning prompt from `docs/02-planning-prompt.md`.
8. Review the implementation plan and confirm the smallest end-to-end slice.
9. Run the build prompt from `docs/03-build-prompt.md`.
10. Build phase by phase instead of asking for the whole app at once.
